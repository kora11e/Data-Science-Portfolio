## [Classification and prediction Model Training with FastAPI](https://github.com/kora11e/Machine-Learning-Fast-API-Modeling)
- Overview: Built backend structure for trained ML models: XGBoost, Random Forest, custom Deep Neural Network with NAS optimization. Trained classification and regression models with hyperparameter tuning. Optimized models with neuron weight pruning. Containerized application backend to Docker, orchestrated with Kubernetes and monitored with Grafana.
- Technologies: Python, FastAPI, XGBoost, PyTorch, Docker, Kubernetes, Grafana

## [Apache Big Data Cosine Similarity Finder](https://github.com/kora11e/Apache-Big-Data-Cosine-Similarity-Calculator)

## [Webscraping with Beautiful Soup](https://github.com/kora11e/Stocks-Webscraper)
- Overview: With BeautifulSoup information about bonds data was collected from [BondSpot.pl](https://www.bondspot.pl/fixing_obligacji?date=20240226&month=02&year=2024). The collected data covers information about 25 top companies from the market and their fixing prices.
- Technologies: Python, BeautifulSoup, Pandas, requests
- Result: [Final dataset](https://github.com/kora11e/Stocks-Webscraper/blob/main/data/bondspot_data_2026.csv). Note that the to-be-gathered information can be easily modified inside the script.
